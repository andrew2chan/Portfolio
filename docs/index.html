Tabletop Games

Tabletop Games is a 3D board and card game simulator. This web app features:

  • The ability to login and register
  • A win and loss tracker
  • Card game war with an AI

In terms of the technology - the MEAN stack was used for tor the front and backend. MongoDB (Mlabs) is used as the database to store login information and lifetime stats. Express is used as the middleware to simplify NodeJS. AngularJS is used for a smooth and seamless SPA experience and lastly, NodeJS is used for the server.

Additionally, for the game itself - we utilized three.js to create the game, scene and animations.

This was an academic project created for my course, CPS630 (Web apps). I was responsible for most of the front and the backend as well as assigning, monitoring and integrating other members code into the main branch and deploying to heroku.
